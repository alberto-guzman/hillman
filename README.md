# Hillman Summer Program Impact Study

Causal evaluation of the Hillman Summer STEM Program on college enrollment,
persistence, and degree attainment. The analysis uses propensity score matching
(PSM) to estimate the Average Treatment Effect on the Treated (ATT) by comparing
program participants to similar non-participants drawn from the applicant pool.

### `code/`

All R analysis scripts. Run in sequence via `run_all.R`.

| Script | Purpose |
|---|---|
| `run_all.R` | Master script — sources all scripts below in order |
| `1_1d_applicant_clean.R` | Load and clean year-specific applicant files (2017–2023) |
| `2_1d_alumni_clean.R` | Load and clean alumni tracker; create treatment indicators |
| `3a_1d_master_merge.R` | Merge applicants to alumni; build per-student treatment timeline |
| `3b_1d_merge_clean.R` | Standardize covariates; merge NSC college outcome data |
| `4_1d_merge_school_info.R` | Normalize school names; merge PA school-level covariates |
| `5_1d_matching.R` | Propensity score matching (nearest-neighbor, caliper = 0.5 SD) |
| `7_1d_impact.R` | ATT estimation via doubly-robust linear probability models |
| `7_1d_impact_subgroup.R` | Subgroup ATT estimates (stipend, race, geography, etc.) |
| `8_1d_tables_figures.R` | Publication-ready tables and figures |
| `cips_list.R` | Build STEM CIP code reference list from DHS/NSF definitions |
| `nsc_clean.R` | NSC outcome cleaning helpers |
| `nsc_do_files/` | Stata do-files used to process raw NSC records (from Danielle) |

To run the full pipeline from RStudio:
```r
source("code/run_all.R")
```

> **Note:** Scripts use the `here` package for paths. Open the `.Rproj` file
> to set the project root before running.

### `data/`

#### `data/raw/`
Original, unmodified input files. Do not edit.

- `applicants/` — Year-specific Hillman applicant files (`hillman_2017.xlsx` through `hillman_2023.csv`) plus stipend reference files
- `alumni/` — Alumni tracker spreadsheets (multiple versions); source of treatment indicators
- `cips/` — CIP code reference files (DHS STEM list, manual curation)
- `nsc/` — Raw NSC (National Student Clearinghouse) enrollment CSV files
- `pa_stats/` — Pennsylvania school-level data from PA DOE (enrollment, demographics)
- Other files — PA school reference (`pa_file.xlsx`), school fast facts (`SchoolFastFacts_20242025.xlsx`), HSLS variable list, HSLS matching reference

#### `data/processed/`
Intermediate datasets produced by the cleaning and merging pipeline. These are
outputs of scripts 1–4 and inputs to the matching step.

Key files:
- `merged_all_states.csv` — Analysis-ready dataset for all states (output of script 4)
- `merged_df_pa_public.csv` — PA public school subsample with school-level covariates
- `clean_Hillman_2017_2025.dta` — Cleaned Stata file with NSC outcomes (from Danielle)
- `high_school_match.dta` — HS name → PA AUN crosswalk
- `hillman_data_for_nsc.xlsx` — Student file submitted to NSC for record matching

#### `data/matched/`
Output of the matching step (script 5). Contains matched datasets and saved
`MatchIt` objects for both analytic samples.

- `matched_all_states_year_only.rds` — Matched data, all states
- `matched_pa_year_only.rds` — Matched data, PA public schools only
- `matchit_object_*.rds` — Saved MatchIt objects (for diagnostics/re-use)
- CSV versions of matched datasets for inspection

#### `data/files_for_danielle_nsc/`
Files exchanged with Danielle (NSC data collaborator). Contains raw NSC CSV
records, Stata do-files for NSC cleaning, and cleaned Stata output files.
Kept as-is for reproducibility and reference.

## Sample and Design Notes

- **Years:** Applicant cohorts 2017–2023; 2020 excluded (COVID disruption)
- **Outcomes:** Seamless enrollment, ever-enrolled, 4-yr enrollment, persistence (1yr), retention (1yr), bachelor's degree (4yr/6yr), STEM enrollment
- **Matching:** Nearest-neighbor PSM, with replacement, caliper = 0.5 SD, exact match on application year
- **Samples:** (1) All states (~2,600 students); (2) PA public schools only (~1,300 students)
- **Estimator:** Doubly-robust linear probability models with HC2 cluster-robust SEs
